# 1. ¿Qué es una Base De Datos Relacional?
es una base de datos que guarda la informacion en forma de tablas,columnas lineas y relacionan los datos mediante claves primarias y externas

# 2. ¿Qué es una Tabla en el contexto Base De Datos Relacionales?
es una estructura organizada en filas y columnas y almacena informacion de un dato en cuestion o especifico

# 3. ¿Qué es una relación en el contexto de Base De Datos Relacionales?
son conexiones logicas con estructuradas para dos o mas tablas estas estan conectadas por medio de campos comunes para garantizar que los datos no se vallan a duplicar y guarden integridad

# 4. ¿Qué representa una columna en el contexto de Base De Datos Relacionales?
una caracteristica que define un campo u espacio de los datos en una tabla

# 5. ¿Qué representa un renglón en el contexto de Base De Datos Relacionales?
un tipo de registro individual en una tabla formando lineas

# 6. ¿Qué es un motor de base de datos relacional?
es el software que crea y gestiona los datos en las tablas filas o columnas usando relaciones ya definidas entre si

# 7. ¿Cuáles son los motores de bases de datos relacionales mas populares?
Oracle Database:es la Líder en el sector empresarial,tambien es conocido por su alta robustez, seguridad y escalabilidad en grandes entornos corporativos lo que hace que su uso sea mas facil.
MySQL: El más popular para desarrollo web y de código abierto por sus caracteristicas. Es el motor por defecto para aplicaciones como WordPress, Drupal y entornos PHP mejorando el rendimiento y la produccion.
Microsoft SQL Server:este es muy popular en entornos empresariales que utilizan tecnología Microsoft, destacando en inteligencia de negocios y siendo multiplataforma permitiendo que mas gente lo use.
PostgreSQL:es conocido por ser código abierto y uno de los más avanzados tecnológicamente, altamente valorado por su cumplimiento de estándares.
